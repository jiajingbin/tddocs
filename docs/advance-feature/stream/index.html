TDengine 是一款高性能、分布式、支持 SQL 的时序数据库，其核心代码包括集群功能全部开源，同时TDengine 还带有内建的缓存、流式计算、数据订阅等系统功能，能大幅减少研发和运维的复杂度，可广泛应用于物联网、车联网、工业互联网、IT运维、金融等领域。
